test.c, unity.c, and csv.c are separate translation units. Only functions with external linkage cross TU boundaries. Calling a static function from test.c fails during linking.

Only fill the buffer in the test using csv_set_field(buffer, row, entry, "text") — this creates rows and fields internally on its own.
