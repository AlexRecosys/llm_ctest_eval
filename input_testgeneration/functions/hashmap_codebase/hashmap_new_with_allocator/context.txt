## hashmap (hashmap.c)
A generic open-addressing hashmap in C using Robin Hood hashing. The central
data structure is struct hashmap (opaque). Items are stored by value (shallow
copy of elsize bytes); the caller supplies hash and compare callbacks that
operate on the key portion of the item struct. Public API functions are
prefixed with hashmap_. Internal helpers are static. The map is not thread-safe.
Pointers returned by hashmap_get(), hashmap_set(), and hashmap_delete() are
invalidated by subsequent map operations callers must copy values out
immediately.
